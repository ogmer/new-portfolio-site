# Ogmer's Portfolio

Next.js で構築されたモダンなポートフォリオサイトです。

## 機能

- ダークモード/ライトモードの切り替え
- プロジェクト一覧の表示
- プロジェクト詳細のモーダル表示
- アニメーション効果（Framer Motion）
- レスポンシブデザイン

## 技術スタック

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## プロジェクト構造

```
src/
  ├── app/
  │   └── page.tsx      # メインページ
  ├── components/
  │   └── Modal.tsx     # モーダルコンポーネント
  └── context/
      └── ThemeContext.tsx  # テーマ管理
```
